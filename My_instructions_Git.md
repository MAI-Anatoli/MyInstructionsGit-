# Инструкция по пройденным командам Git
## Перед тем как использовать Git надо представится Gitu это можно сделать следующим образом:
```sh
git config --global user.name "Name"
```
```sh
git config --global user.email "Email"
```
### После чего представился можно проверить свои данные следующим способом:
```sh
git config user.name
```
```sh
git config user.email
```
## Чтобы перейти в необходимый каталог (папку) нужно использовать следующую команду:
### Для Macos, Linux
```sh
cd ~/Desktop/name_file
```
### Для Windows
```sh
cd C:/user/name/Desktop/file_name
```
## В Macos, Linus можно увидеть весь путь к файлу следующей командой:
```sh
pwd
```
## Листинг текущей директории (эти команды покажут файл или папку с которой вы работаете):
### Для windows:
```sh
dir
```
### Для Macos, Linux:
```sh
ls
```
## С помощью следующей команды можно создать новый репозиторий (скрытая папка для работы) ***Git***.
```sh
git init
```
## Показывает состояние рабочего каталога и проиндексированного снимка состояния ( показывает изменения в файле с которым вы работаете)
```sh
git status
```
## Переносит изменения из рабочего каталога в раздел проиндексированных файлов
```sh
git add
```
## Получает проиндексированный снимок состояния и выполняет его коммит в историю проекта (комментирует проиндексированы файл для облегчения работы с проектом и ориентацией в нем)
```sh
git commit -m "text"
```
## Чтобы удалить ненужный файл воспользуемся следующими командами:
### Для Windows
```sh
del <name_file>
```
### Для Linux, Macos
```sh
rm <name_file>
```
